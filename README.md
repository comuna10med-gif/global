
# Recovery Agency Template

## Instalación
```bash
npm install
npm run dev
```

## Despliegue en Vercel
1. Sube este proyecto a GitHub
2. Entra a Vercel
3. Importa el repositorio
4. Deploy

## Personalización
- Cambia el número de WhatsApp en `app/page.js`
- Modifica textos e idiomas fácilmente
- Reemplaza imágenes en `/public`
